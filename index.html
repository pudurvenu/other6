Website Controlled Home Automation Simulation

Power outages and unstable power supplies are becoming increasingly common, necessitating the acquisition of a continuous block of data to operate appliances from remote locations. The home automation is the most efficient approach to maintain track of input data and provide the necessary service. Tinkercad, an Autodesk simulation tool, is used in this project to imitate (simulate) the whole home automation with the greatest possible control. In order to discreetly organise the things at work or at home and their control to allow them work for us i have used website as a controlling medium.

This home automation project simulation is based on the identical model found in the actual prototype. The Arduino UNO is used as the project's controller, and the ESP8266 wifi module is utilised to send HTTP requests and receive commands to direct the Arduino as needed.

The Arduino is programmed in C using the Arduino IDE in this project. This project allows you to manage your lights, fans, and other home appliances. For data storage, an HTTP call is made from Arduino to the Thingspeak Cloud Platform server. Data is retrieved from Thingspeak using Javascript and interactive details are shown on a website. The peripheral devices interfaced to the Arduino are configured to collect input data from physically interacting devices with different physical values.

Thingspeak is a cloud platform that stores real-time data from Arduino and displays it graphically and digitally. During the development of a website, the API documentation of the Thingspeak platform is consulted. A fully responsive website has been created to make it user-friendly and controllable. In order to display them as a userfriendly, a digital panel for electrical appliances is built, and a type of variable electrical parameter regulating function is included in comparable to the physical components. Even those who are uneducated and non-technical can readily operate them (appliances) from a far.
